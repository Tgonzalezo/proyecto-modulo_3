# Sistema de Gestión de Inventario para Supermercado

## Descripción

Este proyecto consiste en un **Sistema de Gestión de Inventario para un Supermercado**, desarrollado en **Python** como parte de la evaluación del módulo 3 de programación.

El sistema permite administrar el inventario de productos de un supermercado mediante un menú interactivo en consola, facilitando el registro, consulta, actualización y organización de la información de los productos.

Además, implementa técnicas de programación como modularización, funciones, estructuras de datos, recursividad y expresiones `lambda`, permitiendo un código más organizado y eficiente.

---

## Funcionalidades

El sistema permite realizar las siguientes operaciones:

* Agregar nuevos productos al inventario.
* Mostrar todos los productos registrados.
* Buscar un producto mediante su código.
* Actualizar el stock de un producto.
* Eliminar productos del inventario.
* Aplicar descuentos al precio de un producto.
* Ordenar los productos alfabéticamente.
* Mostrar las categorías existentes en el inventario.
* Calcular el valor total del inventario mediante una función recursiva.
